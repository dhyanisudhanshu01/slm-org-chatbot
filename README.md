# slm-org-chatbot

This project is a **fully open-source, zero-cost organization chatbot**
designed to run on **low-resource systems** while ensuring **company data security**.

## Key Features
- Uses **Small Language Models (SLMs)**
- No paid cloud services
- No OpenAI / Gemini / proprietary APIs
- CPU-only deployment
- Retrieval-Augmented Generation (RAG)
- Based on **Bitext Customer Support Dataset**
- Deployed on **Hugging Face Spaces**
